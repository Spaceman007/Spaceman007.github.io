用hexo搭建個人博客

Posted on 2018-01-23 | Edited on 2018-07-15 | Comments:

使用

1. 常用命令

命令 全寫 含義
hexo g hexo generate 生成靜態文件
hexo s hexo server 啓動本地web服務，用於預覽
hexo d hexo deploy 部署（到github pages等）
hexo new ‘postName’ 新建文章
hexo new page ‘pageName’ 新建頁面

2. 配置說明（_config.yml）

2.1 sitemap

要讓搜索引擎搜索到我們的博客，需要添加 sitemap.

如何生成 sitemmap?

只需安裝hexo-generator-sitemap 和 hexo-generator-baidu-sitemap 即可

之後，當我們 hexo g 時，就會在 public 目錄下生成 sitemap.xml 和 baidusitemap.xml

提交鏈接

點擊 百度链接提交，將網址提交給百度。
